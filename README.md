# Basic Node.js Activity

This repository contains a basic Node.js activity demonstrating how to create a simple web server using Node.js and Express.js.

## Instructor
Wilson Gayo

## Requirements
- Node.js
- npm (included with Node.js)

## Installation
Open a terminal or command prompt in the project folder and run:

npm install

## Running the Application
Start the server with:

node index.js

Once the server is running, open your browser and go to:

http://localhost:3000
